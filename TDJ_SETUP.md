# Tie Dye Jedi Developer Setup Guide

Welcome to the Tie Dye Jedi coding team!

This guide will walk you through preparing your computer and connecting to the team's robot code repository.

## Required Software

### 1. Install Android Studio
Android Studio is the official development environment used by FIRST Tech Challenge teams.

Download:
https://developer.android.com/studio

### 2. Install Git

Download:
https://git-scm.com/downloads

### 3. Create a GitHub Account

Create a free account:
https://github.com

Provide your GitHub username to the coaching staff so you can be granted access to the repository.

## Clone the Repository

Once you have access:

1. Clone the repository.
2. Open the project in Android Studio.
3. Allow Gradle to sync.
4. Build the project.

The build should complete successfully before proceeding.

## FTC Development Workflow

The robot code lives primarily in:

TeamCode/

This is where students will create and modify robot code.

## First Coding Exercise

After setup is complete, proceed to Workbook 01:
"Get a Motor Moving"

Goals:
- Connect to the robot
- Deploy code
- Control a motor
- Understand the FTC software architecture

## Need Help?

Ask questions early.

Nobody is expected to know everything.

The goal is to learn, experiment, and improve together.
